## Deployment

> [!WARNING]
> You **can not** host Infrared locally via a static web hosting site
> such as Netlify, Github Pages, and Cloudflare Pages.

Please run the following commands to host a server:

```
git clone https://gitlab.com/nightnetwork/infrared.git
cd Infrared
pnpm i
pnpm start
```

### Updating

```
git pull --force --allow-unrelated-histories
```

## Contribution

To contribute to Infrared, clone the repo locally and submit your code in a pull
request. If you are making large changes, please make sure you test your code
before submitting a pull request. Please check [contributing.md](https://gitlab.com/nightnetwork/infrared/blob/main/contributing.md) for a more in-depth explanation.
